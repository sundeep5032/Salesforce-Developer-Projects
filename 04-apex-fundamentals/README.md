📘 STATEMENTS • EXPRESSIONS • OPERATORS 
========================================
🔹 SECTION 1: Expressions
--------------------------
An expression produces a value.
It can be evaluated to something (number, boolean, string, object).

10 + 5
7 + 8
13 + 2

🔹 SECTION 2: Statements
--------------------------
A statement performs an action.
Usually ends with a semicolon (;).

Integer x = 10;
System.debug(x);

🔹 SECTION 3: Statement vs Expression (Key Difference)
-------------------------------------------------------
Expression{
    Produces a value,
    Can be part of a statement,
    Example: a + b
}
Statement{
    Performs an action,
    Cannot exist inside another statement,
    Example: Integer c = a + b;
}
